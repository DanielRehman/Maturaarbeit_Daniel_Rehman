# Optionale große Run-Imports

Dieser Ordner ist nicht für das normale Setup nötig.

Die SQL-Datei enthält historische Runs, Scores, Checkpoint-Ergebnisse und feste Paarbeziehungen. Sie ist groß und wird nur importiert, wenn die alten Auswertungsdaten benötigt werden.

Import:

```bat
importiere_optionale_runs.bat
```

Für eine kleine, leere Anwendung kann dieser Ordner weggelassen werden.
